## 🛠️ 构建指南 (Build Instructions)

想要亲手捕捉这只“云音乐猫”吗？按照以下步骤操作就可以啦，喵！

### 1. 前置要求

首先，请确保你的电脑里住着 **Node.js 22** 版本。你可以通过以下命令检查喵：

```bash
node -v  # 确认输出为 v22.x.x

```

### 2. 获取与准备

打开你的终端（或者命令提示符），跳进项目的怀抱：

```bash
# 进入项目路径
cd NekoMusicForPC

```

### 3. 安装依赖

让猫猫吃饱（安装必要的库文件）：

```bash
npm install

```

### 4. 执行构建

可以根据正在使用的平台，选择对应的构建指令喵：

| 目标平台 | 执行指令 |
| --- | --- |
| **Windows** | `npm run build:win` |
| **Linux** | `npm run build:linux` |
| **macOS** | `npm run build:mac` |

---

## 📂 构建产物

构建成功后，你可以在项目根目录下的 `dist` (或指定的输出文件夹) 中找到打包好的可执行文件喵！
